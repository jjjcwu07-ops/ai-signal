Aaron Levie @levie 2026-06-19
The main variable in getting success with agents is whether you can get the agent the context it needs to do its work; and a major factor in that is if you can create a shared work… ↗ 查看原推
中文摘要

Box CEO Aaron Levie 认为，决定 Agent 成败的关键变量只有一个：Agent 能否获取它所需的上下文。而实现这一点的核心，是创建一个人与 Agent 都能理解、共用的工作空间。

他以文件系统为例展开说明：Agent 直接使用文件系统来处理数据，意味着人与 Agent 可以在同一套体系内协作传递信息——计划、笔记、任务列表、策略、草稿、日志、修正、决策等，都放在同一个"文件系统界面"里。这个界面既对模型可读，也对人类可理解。

本质上，Levie 想表达的是：把 Agent 接入我们已经熟悉的工作系统，但为它们做最优化的适配，这才是 Agent 最理想的运行基元。
